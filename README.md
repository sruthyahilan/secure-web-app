# secure-web-app
# Secure Cloud-Based Web Application (DevSecOps)

## Project Description
This project is a secure cloud-based web application developed using DevSecOps practices.

The application is deployed on AWS cloud infrastructure and includes:
- Secure user authentication
- Reverse proxy using Nginx
- HTTPS encryption
- Docker containerization
- CI/CD automation
- Monitoring and logging
- Cloud storage using AWS S3
- Database using AWS RDS

The goal of this project is to demonstrate practical implementation of:
- AWS Cloud
- Linux Administration
- Cyber Security
- DevOps Automation
- Monitoring & Logging
